ADD/UPDATE A PROJECT
--------------------

1. add a folder in the "projects" folder to create a new project

-> the order of the projects is based on the names of the folder, arranged alphabetically

-> titles of folders should only contain a-z, A-Z, 0-9, hyphen and underscore

2. put the project images in the folder (jpg, gif, or png)

-> images should not be heavier than 800 KB

3. rename the .html file with the height of the images

4. write the description of the project in the content of the html file

-> tags <a href="URL">for a link</a>, <em>for italic</em> and <strong>for bold</strong>

UPDATE ABOUT
------------

5. write the content of the about page in the "content.html" file in the "about" folder

-> tags <a> for a link, <em> for italic and <strong> for bold

UPLOAD WEBSITE ONLINE
---------------------

1. Make sure you have properly completed all the updates

2. Generate an HTML "static" version of the website

-> create a local server by opening the terminal, type "php -t /Users/markemil/Documents/GitHub/Markemilpoulsenwebsite -S localhost:8000" and press Enter

-> on the browser, go to http://localhost:8000, save the page content as index.html in the folder "Markemilpoulsenwebsite"

-> if you have made updates on the about page, save the "page source" of the about page as index.html in the folder "Markemilpoulsenwebsite/about"

3. On GitHub Desktop, "commit" the updates and "push" them to GitHub

4. Ask your friend to "deploy" the website

ADDITIONAL INFO
---------------

6. a direct link to any specific project is markemilpoulsen#folder-name
